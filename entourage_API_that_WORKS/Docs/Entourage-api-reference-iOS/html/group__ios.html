class GnAudioSourceiOSMic

The GnAudioSourceiOSMic class implements a simplified interface to configure and control the iOS audio input device for use with Gracenote SDKs. This is a convenience class and may not be suitable for every application. This class does not assume or take responsibility for mic input interruptions or route changes. This is the responsibility of the application. To learn more about handling these types of interruptions, familiarize yourself with the AVAudioSession documentation from Apple. http://developer.apple.com/library/ios/#documentation/Audio/Conceptual/AudioSessionProgrammingGuide/Introduction/Introduction.html

- (id) initWithAudioConfig: (GnAcrAudioConfig *)  config

A GnAudioSourceiOSMic object must be initialized with an GnAcrAudioConfig object. This config should repesent the same config that is needed to feed the writeBytes APIs of GnAcr or GnMusicID.

- (id) initWithAudioConfig: (GnAcrAudioConfig *)  config
error: (NSError **)  error 

A GnAudioSourceiOSMic object must be initialized with an GnAcrAudioConfig object. This config should repesent the same config that is needed to feed the writeBytes APIs of GnAcr or GnMusicID. This API returns an error in case there is any error while using iOS audio APIs.

- (NSError*) start

Activates audio session and starts the stream of audio data from the device inputs (microphone) through the delegate callback.

Deprecated:
This method is deprecated in version 1.5 and later
- (NSError*) startMicrophone

Starts the stream of audio data from the microphone. Note: The audio session is owned by the applicaiton.

- (NSError*) stop

Deactivates audio session and stops the stream of audio data from the device inputs (microphone) through the delegate callback.

Deprecated:
This method is deprecated in version 1.5 and later
- (NSError*) stopMicrophone

Stops the stream of audio data from the microphone. Note: The audio session is owned by the applicaiton.
