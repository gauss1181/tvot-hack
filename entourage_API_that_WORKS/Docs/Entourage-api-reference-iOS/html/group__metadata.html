class GnResult

GnResult is the universal reponse object for all SDK queries. It encapsulates all metadata objects returned from a specific "find" API and the IGnAcrResultDelegate callback. To access the indivudual results for a query, iterate over the NSEnumerator of the appropriate result type. Typically, only one NSEnumerator is valid for a given result. This will depend on the "find" API from which the GnResult object was returned.
